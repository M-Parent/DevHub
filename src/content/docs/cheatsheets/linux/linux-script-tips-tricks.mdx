Enhance your terminal productivity by adding custom aliases and functions to your shell configuration. These snippets help automate repetitive tasks like Git commands, directory navigation, and file extraction.

## Configuration

To add these, edit your bash configuration file:

```bash
nano ~/.bashrc
```

Paste your shortcuts at the end of the file, save, and exit.

---

## Useful Aliases

### Git Shortcuts
Speed up your Git workflow with these simple aliases.

```bash
# Add and commit in one command
alias ac='git add . && git commit -m'

# Push to main branch
alias gp='git push origin main'
```

### Navigation
Stop typing `cd ..` repeatedly.

```bash
alias ..='cd ..'
alias ...='cd ../..'
alias ....='cd ../../..'
alias .....='cd ../../../..'
```

## Powerful Functions

Functions allow you to pass arguments, making them more flexible than standard aliases.

### Create and Enter Directory
Instead of `mkdir folder && cd folder`, use one command.

```bash
mkcd () {
  mkdir -p "$1" && cd "$1"
}
```

### Universal Extractor
Never struggle with `tar` or `unzip` flags again. This function handles almost any compressed format.

```bash
extract () {
  if [ -f $1 ] ; then
    case $1 in
      *.tar.bz2)   tar xjf $1     ;;
      *.tar.gz)    tar xzf $1     ;;
      *.bz2)       bunzip2 $1     ;;
      *.rar)       unrar x $1     ;;
      *.gz)        gunzip $1      ;;
      *.tar)       tar xf $1      ;;
      *.tbz2)      tar xjf $1     ;;
      *.tgz)       tar xzf $1     ;;
      *.zip)       unzip $1       ;;
      *.Z)         uncompress $1  ;;
      *.7z)        7z x $1        ;;
      *)           echo "'$1' can't be extract via extract()" ;;
    esac
  else
    echo "'$1' is not a valid file"
  fi
}
```

## Applying Changes

After saving `~/.bashrc`, you must reload the file for the changes to take effect in your current session.

<Aside type="tip">
  You only need to do this once per terminal session after editing the file.
</Aside>
