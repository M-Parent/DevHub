## Docker Desktop Setup Checklist

<Steps>

1. **Enable Virtualization in BIOS**
   - Usually found under CPU or Advanced settings
   - Required for Docker to run containers efficiently

2. **Download and run the Docker Desktop installer**
   - Download the `.exe` installer from Docker Hub
   - Run it and follow the setup wizard

3. **Enable WSL 2 (Windows Subsystem for Linux)**

   Run PowerShell as Administrator:

   ```powershell
   wsl --install
   ```

   Reboot your system if prompted.

4. **Link Docker Desktop with WSL 2**
   - Open Docker Desktop â†’ **Settings â†’ General**
   - Ensure "Use the WSL 2 based engine" is checked
   - Under **Resources â†’ WSL Integration**, select which distros Docker can access

5. **Verify the installation**

   Open a terminal (CMD or PowerShell) and run:

   ```bash
   docker version
   ```

   <Aside type="tip">
     If you see a Docker version number in the terminal output, Docker is
     successfully installed!
   </Aside>

</Steps>
